# johnnygreco.dev

Personal website for Johnny Greco. Built with Astro 6, Tailwind v4, deployed to GitHub Pages.

## Quick start

```bash
npm install
npm run dev       # http://localhost:4321
```

## Scripts

| Command | Action |
|---------|--------|
| `npm run dev` | Start dev server with hot reload |
| `npm run build` | Production build + Pagefind search index |
| `npm run preview` | Preview production build locally |
| `npm test` | Run E2E tests (Playwright) |
| `npm run validate` | Build + analyze + test everything |
