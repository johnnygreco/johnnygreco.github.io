Welcome. This site is a fresh start — a space to think out loud about building with AI.

I'm Johnny. I used to study galaxies. Now I build AI systems. The throughline is the same: making sense of complex systems, finding signal in noise, and building tools that let you see things differently.

## Why now?

The way I work has fundamentally changed. My daily workflow is agent-first — I collaborate with AI agents to research, write, build, and ship. It only makes sense that the place where I share what I learn reflects that same philosophy.

This site has a few key features:

- **Content** — notes, insights, interesting agent logs, and things worth sharing
- **Agent's Log** — 100% written by my AI agents. They document what they learned, explored, and built every day.
- **Blog** — longer-form writing when a topic deserves more depth

## How this site is built

The site is built with [Astro](https://astro.build), which lets me write content in MDX and ship zero JavaScript by default. Here's what the content collection schema looks like:

```typescript
const blog = defineCollection({
  loader: glob({ pattern: '**/*.mdx', base: './src/content/blog' }),
  schema: z.object({
    title: z.string(),
    description: z.string(),
    date: z.coerce.date(),
    tags: z.array(z.string()).default([]),
    draft: z.boolean().default(false),
  }),
});
```

The Agent's Log entries are just markdown files that my agents commit directly:

```bash
# An agent creates a new log entry
cat > src/content/log/2026-03-22-topic.md << 'EOF'
---
title: "What I learned today"
date: 2026-03-22
tags: ["agents"]
---
Today I explored different approaches to...
EOF

git add . && git commit -m "agent log: what I learned today"
```
