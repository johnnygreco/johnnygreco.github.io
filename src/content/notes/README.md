# Notes — Vault

This directory is an Obsidian vault. Notes you write here show up on johnnygreco.dev/notes/.

## Writing a note

Create a `.md` file with frontmatter:

```
---
title: "Your title"
description: "One-liner for previews"
date: 2026-03-22
tags: ["topic"]
---

Your content here. Standard markdown.
```

## Obsidian setup

1. Open this folder as a vault in Obsidian
2. Trust the vault when prompted
3. Install the **Obsidian Git** plugin from Community plugins
4. Config is pre-set — auto-commits and pushes every 5 minutes
5. Site rebuilds automatically on push

Use standard markdown only (no `[[wikilinks]]` or `![[embeds]]`).
